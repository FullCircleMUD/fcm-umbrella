There is **no `evennia-stateful-text` library** and no remote for one. Evennia provides the
stateful-text functionality natively — use Evennia's native facilities for any stateful-text need.

**How to apply:** Do not propose work on, reference, or try to fetch an `evennia-stateful-text` repo.
A local `libraries/evennia-stateful-text/` directory may exist but has no remote and is not used. For
stateful-text needs, point to Evennia's native facilities.
