The evennia-stateful-text library is no longer a piece of work. Investigation revealed that Evennia provides the required functionality natively, so the GitHub repo was deleted.

**Why:** Native Evennia functionality made the standalone library redundant.

**How to apply:** Do not propose work on, reference, or try to fetch the evennia-stateful-text GitHub repo. A local directory may still exist at `libraries/evennia-stateful-text/` but it has no remote. If suggesting libraries for stateful-text needs, point to Evennia's native facilities instead.
